Question-5: How do you create, add, and remove elements using JavaScript?

Ans:

Create an element= document.createElement("tagName")
Add as last child= parent.appendChild(child)
Add as first child= parent.prepend(child)
Insert before a specific element= parent.insertBefore(newElement, referenceElement)
Remove an element directly= element.remove()
Remove a child from parent= parent.removeChild(child)
Remove all children= parent.innerHTML = ""
